# KeySentinel Frontend

Next.js 14 frontend for KeySentinel — Developer API Key Vault & Secrets Management.

## Stack
- Next.js 14 (App Router)
- Tailwind CSS
- shadcn/ui components
- TypeScript

## Features
- Centralized key vault dashboard
- Project-based access scoping UI
- Key health audit report view
- Rotation workflow tracker UI
- Expiry & rotation reminder settings
- Slack/webhook alerting configuration

## Deployment
Deploy to Vercel using vercel.json
